# GroupWebShop

GroupWebShop is our first collaborative webshop project in the Frontend course.  
We are building it using **HTML**, **CSS**, and **JavaScript**, and the site will be deployed through **GitHub Pages**.

The webshop will feature a variety of products such as clothing, electronics, accessories, and more.
